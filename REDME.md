# Java Simple Calculator

## Project Description
This project is a simple command-line calculator application developed using Java. 
The program allows users to perform basic arithmetic operations such as addition, 
subtraction, multiplication, and division by entering numbers and selecting the 
desired operation.

This project is created to practice fundamental Java programming concepts such as 
user input handling, conditional statements, and basic program structure.

---

## Features
- Perform Addition
- Perform Subtraction
- Perform Multiplication
- Perform Division
- Simple and easy-to-use command line interface

---

## Technologies Used
- Java
- Command Line Interface (CLI)

---

## How the Program Works
1. The program asks the user to enter two numbers.
2. It displays a list of arithmetic operations.
3. The user selects an operation.
4. The program performs the calculation.
5. The result is displayed on the screen.

---

## How to Run the Program

### Step 1: Compile the Java program
           javac Calculator.java
### step 2:Run the program
           java Caluculator
Example:
Enter first number: 10 Enter second number: 5
Add
Subtract
Multiply
Divide
Choose operation: 1
Result: 15
